# ギャラリーサイト コーディング仕様

## 基本設定

| 項目 | 値 |
|------|-----|
| フォント | Google Fonts「Source Sans Pro」 |
| コンテンツ幅 | 800px |
| サイドパディング | 30px |
| ブレイクポイント | 900px |

---

## 各セクション仕様

### Header

- ロゴとハンバーガーメニューを `position: fixed` で両端に固定
- 初期状態は `display: none` で非表示
- スクロール位置が**520px**に達した時に表示（jQuery fadeIn使用）
- スクロール位置は `$(window).scrollTop()` で取得

#### ハンバーガーメニュー

- 3本線を表現するクラスを作成
- クリック時に全画面メニューを上から下へスライド表示
- メニューは nav、ul、li タグで記述し、初期状態は非表示
- jQueryでオープン/クローズを切り替え

---

### メインビジュアル（#mainvisual）

- 画像をFlexboxで横並び配置
- `overflow-x: hidden` で横スクロールを防止

#### PC表示

- スクロールで画像を**拡大**
- jQueryのスクロールイベントで `#mainvisual img` の width を変更
- width値はスクロール量に応じて増加（`$(window).scrollTop()` を使用して計算）

#### SP表示

- スクロールで画像を**縮小**
- width値はスクロール量に応じて減少

---

---

### INFORMATION（#information）

#### 要素構成

- sectionタグで囲む
- h2タグでタイトル
- pタグで各テキスト（フォントサイズやカラーが異なる場合はspanタグ使用）

#### アニメーション

- 要素が**画面下にきたタイミング**で下からふわっと表示
- jQueryプラグイン「**inview**」を使用
- 配布元: [https://github.com/protonet/jquery.inview](https://github.com/protonet/jquery.inview)
- `jquery.inview.min.js` をダウンロードして作業ディレクトリに配置

---

### GALLERY（#gallery）

#### 要素構成

- sectionタグで囲む
- h2タグでタイトル
- ul、liタグで画像を記述

#### レイアウト

- 画像を交互に左右に配置
- 奇数のli: `text-align: left`（nth-child(odd)で指定）
- 偶数のli: `text-align: right`（nth-child(even)で指定）

#### アニメーション

- 要素が**画面下にきたタイミング**で下からふわっと表示
- jQueryプラグイン「inview」を使用

---

### サイドボタン（PC表示のみ）

- **GALLERYタイトル**が画面下にきたタイミングで右から左へスライド表示
- 右下に固定配置
- **ACCESSタイトル**が画面下にきたタイミングでスライドして非表示
- スマホでは非表示（`display: none`）

---

### ACCESS（#access）

#### 要素構成

- sectionタグで囲む
- h2タグでタイトル

#### 背景画像の切り替え

- **ACCESSタイトル**が画面下にきたタイミングで背景画像を表示（jQuery fadeIn）
- 背景画像用クラスの設定:
  - `position: fixed` で固定
  - `width: 100vw` と `height: 100vh` で全画面表示
- **CONTACTタイトル**が画面下にきたタイミングで背景画像を非表示（jQuery fadeOut）

---

### CONTACT（#contact）

#### 要素構成

- sectionタグで囲む
- h2タグでタイトル

#### ボタン

- 矢印部分は疑似要素（before、after）で設定
  - before: 直線
  - after: 矢印の先端

#### ホバー時（PCのみ）

- 透過をかける
- 矢印を右へ移動（`right` の値を変更）
- スマホの場合は矢印を非表示

---

### Footer

#### 要素構成

- footerタグで囲む
- フッターメニュー:
  - 左側: divタグ
  - 右側: ul、liタグ
  - Flexboxで横並び配置

---

## レスポンシブ対応

```css
/* PC */
@media (min-width: 901px) {
  /* PC用スタイル */
}

/* SP */
@media (max-width: 900px) {
  /* SP用スタイル */
}
```

## 使用技術・プラグイン

### jQuery プラグイン

- **inview**: スクロール時の要素表示判定
- 配布元: [https://github.com/protonet/jquery.inview](https://github.com/protonet/jquery.inview)
- ファイル名: `jquery.inview.min.js`

### Google Fonts

- フォント: **Source Sans Pro**

---

## 実装のポイント

### スクロールイベント

- `$(window).scrollTop()` でスクロール位置を取得
- 特定の位置（520px）で要素を表示
- スクロール量に応じた画像の拡大/縮小

### 要素の表示タイミング

- 「要素が画面下にきたタイミング」は inview プラグインで判定
- fadeIn / fadeOut で表示/非表示を切り替え

### 横スクロール防止

- メインビジュアルの拡大時に `overflow-x: hidden` で制御
